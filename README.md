## 🛠️ Jak uruchomić projekt
1. Pobierz oficjalny zbiór danych bezpośrednio z Kaggle: [Brazilian E-Commerce Public Dataset by Olist](https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce).
2. Rozpakuj archiwum i umieść wszystkie pliki `.csv` w głównym katalogu tego projektu (obok notebooka).
3. Otwórz i uruchom plik `Analiza_Olist_EDA.ipynb`.
